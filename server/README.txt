In order to run this server on your machine, Node must be installed.
Node can be found at: https://nodejs.org/en/download/

The Mongod service must also be installed.
mongoDB can be found at: https://www.mongodb.com/download-center/community?jmp=docs

To install dependencies, run the following lines in command prompt

npm install express
npm install mongoose

To start server

node server.js

The server will automatically populate with dummy data if it does not exist.
